# Gemini Live API Skill

Documentation and best practices for low-latency streaming and real-time interactions.

## Key Features
- **WebSocket connections** for low-latency streaming.
- Streaming audio, video, and text.
- **Voice activity detection (VAD)** and barge-in support.

## Integration Patterns
- Use this skill for applications requiring real-time voice or multimodal feedback.
- Focus on efficient WebSocket management and handling concurrent media streams.
