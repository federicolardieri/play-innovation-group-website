# Gemini Interactions API Skill

The Interactions API is a unified interface for interacting with Gemini models and agents, designed for agentic applications.

## Key Capabilities
- Text generation, multi-turn chat, and streaming.
- **Function calling**, structured output, and image generation.
- Background execution and **Deep Research** agents.
- Server-side conversation state management.
- Python and TypeScript SDK patterns.

## Recommended Usage
Use this skill when manual control over agentic behaviors is required, such as managing long-running tasks or integrating with complex tools.
